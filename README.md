# 🔐 CLI 會員系統 - 使用 Python、MySQL、bcrypt

## 📌 專案簡介
這是一個以命令列介面（CLI）為基礎的會員系統，提供使用者註冊與登入功能，
後端使用 MySQL 資料庫，並透過 `bcrypt` 實作密碼加密，強化帳號資訊的安全性。

---

## 🛠️ 技術與工具
- Python：撰寫主程式與資料庫邏輯
- MySQL：使用者資料儲存
- PyMySQL：連接 Python 與 MySQL
- bcrypt：密碼加密與驗證

---

## 🚀 功能列表
- ✅ 使用者註冊（避免帳號重複）
- ✅ 密碼使用 `bcrypt` 加密儲存
- ✅ 使用者登入（加密密碼比對）
- ✅ 自動建立資料庫與資料表
- ✅ CLI 互動式選單介面
